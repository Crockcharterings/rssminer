Firebug 1.9b2

25 November 2011 | 5:50 pm

Firebug 1.9b2 has been released.

At this point (beta phase) we are focusing mainly on:

  • Stability and regression bugs
  • Memory Leaks

Let us know if Firebug causes higher memory consumption. Several leaks have been already fixed and so, we expect improvement in this area!

We would also like to encourage all Firebug users to download Firebug releases from addons.mozilla.org (AMO). Its infrastructure is better and you can also get automated Firefox compatibility updates with new Firefox versions.

The existing getfirebug.com/releases archive will be still maintained since it represents great history of all releases (not only Firebug but also some in-house developed extensions).

In any case, we are planning to redirect all Firebug updates to AMO. So, even if Firebug is downloaded from getfirebug.com updates will come from AMO.

All this stuff is transparent so, regular Firebug user shouldn’t even notice that.

Firebug 1.9b2 fixes 10 issues.

Post feedback in the newsgroup, thanks!
